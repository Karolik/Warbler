# Warbler
Full-stack web application in Node, React, Redux.

Deployed at: https://warbler-client-kg.herokuapp.com/

## Getting started
To open it locally:

1.  In the `warbler-server` folder

    * `npm install`
    * `nodemon` (if installed) or `node index.js`

2.  In the `warbler-client` folder

    * `npm install`
    * `npm start`
